Explicación de las Principales Etiquetas HTML Utilizadas en la Creación del Website

<!DOCTYPE html>

Declara el tipo de documento y la versión de HTML que se está utilizando.

<!DOCTYPE html>

<html>

La etiqueta raíz que envuelve todo el contenido HTML.

<html lang="es">

<head>

Contiene metadatos, enlaces a hojas de estilo, y el título del documento.

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mi Proyecto Web</title>
  <link rel="stylesheet" href="css/styles.css">
</head>

<body>

Contiene todo el contenido visible de la página web.

<body>
  ...
</body>

<header>

Define un encabezado para un documento o sección, y usualmente contiene elementos como el logo, el menú de navegación, y el título del sitio.

<header>
  <h1>Bienvenido a Mi Proyecto Web</h1>
  <nav>
    <ul>
      <li><a href="index.html">Inicio</a></li>
      ...
    </ul>
  </nav>
</header>

<nav>

Define una sección de navegación con enlaces a otras partes del sitio web.

<nav>
  <ul>
    <li><a href="index.html">Inicio</a></li>
    ...
  </ul>
</nav>

<main>

Contiene el contenido principal del documento.

<main>
  <section>
    <h2>Introducción</h2>
    <p>Bienvenido a mi proyecto web...</p>
  </section>
</main>

<section>

Define una sección en el documento.

<section>
  <h2>Conceptos Básicos de Seguridad Informática</h2>
  <div class="image">
    <img src="Imagenes/imagen3.jpg" alt="Conceptos Básicos de Seguridad Informática" style="width: 400px; height: auto;">
    <div class="text">
      <h3>Conceptos Básicos de Seguridad Informática</h3>
      <p>Ilustración que representa los conceptos básicos de seguridad informática...</p>
    </div>
  </div>
</section>

<h1>, <h2>, <h3>

Encabezados que definen títulos y subtítulos de diferentes niveles.

<h1>Bienvenido a Mi Proyecto Web</h1>
<h2>Conceptos Básicos de Seguridad Informática</h2>
<h3>Ilustración que representa los conceptos básicos...</h3>

<p>

Define un párrafo.

<p>Bienvenido a mi proyecto web. En este sitio, exploraremos varios temas relevantes sobre tecnología y seguridad informática...</p>

<img>

Incorpora una imagen en el documento.

<img src="Imagenes/imagen3.jpg" alt="Conceptos Básicos de Seguridad Informática" style="width: 400px; height: auto;">

<a>

Define un hipervínculo para enlazar a otras páginas o recursos.

<a href="https://www.youtube.com/watch?v=uFTATNNgHfA" target="_blank">Ver video sobre seguridad informática</a>

<footer>

Define un pie de página para el documento o sección.

<footer>
  <p>© 2024 Mi Sitio Web</p>
</footer>
